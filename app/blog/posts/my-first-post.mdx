# 🚀 How I Built My Portfolio

Welcome to my dev playground — where creativity meets code.

## 🧠 Tech Stack
- **Framework**: Next.js with App Router
- **Styling**: Tailwind CSS + custom variants
- **Animations**: Framer Motion
- **Deployment**: Vercel (free + fast)

## 🎨 Design Highlights
- Hero, About, Projects, Blog, Testimonials & Contact
- Blog system using MDX with metadata and thumbnails
- Testimonials and contact form fully integrated

## 🔐 Security Features
- `/work` page protected via password
- Robots and sitemap generated via Next.js

## 📦 Tools I Used
- GitHub
- Google Cloud Shell
- NDVI satellite analysis for real Earth data
